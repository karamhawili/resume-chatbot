# Karam Abou Hawili - Digital Resume with AI Chat

A Next.js digital resume site with an AI-powered chat assistant that answers questions about the resume.

## Live Demo
https://resume-chatbot-ecru.vercel.app/

## Features
- Professional resume display
- AI chat assistant using OpenAI
- Responsive 2-panel layout
- Built with Next.js 14, TypeScript, and CSS Modules

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- OpenAI API (GPT-3.5-turbo)
- CSS Modules
- Deployed on Vercel

## Running Locally
1. Clone the repository
2. Install dependencies: `npm install`
3. Create `.env.local` with `OPENAI_API_KEY=your_key_here`
4. Run dev server: `npm run dev`
